# 🍽️ 식당 예약 서비스

## 📌 소개  
React와 Flask를 활용한 **웹 기반 식당 예약 서비스**입니다.  
사용자는 예약 가능한 테이블 현황을 간편하게 확인하고, 식당 예약을 진행할 수 있습니다.

## 🛠️ 기술 스택
- **Frontend**: React
- **Backend**: Flask
  
## 🚀 주요 기능
- 레스토랑의 예약 가능한 테이블 현황 확인
- 원하는 날짜, 시간, 테이블 종류 선택하여 예약
- 예약 현황 확인
- 원하지 않는 예약 삭제
